Este site foi criado para oferecer uma experiência acessível e dinâmica.

O que é acessibilidade?

A acessibilidade digital garante que todos, independentemente de limitações, possam navegar e interagir com a web.

Dicas de inclusão

  • Use legendas em vídeos.
  • Adote leitores de tela para textos.
  • Garanta contrastes adequados.

Ferramentas úteis

Confira algumas ferramentas para tornar a web mais acessível:
